== Installation ==

It's really easy and super simple to install **NextCode Options Framework - Lite** plugin.

= Automatic Installation =
1.  Go to `Plugins > Add New` screen in WordPress.
2.  Search for `NextCode Options Framework - Lite`.
3.  Install and activate the plugin, that's it.

= Manual Installation =
1.  Download [DeskAddons](https://downloads.wordpress.org/plugin/ncode-options-framework-lite.zip "Download NextCode").
2.  Extract the `ncode-options-framework-lite.zip` file. You'll get plugin files inside `NextCode` directory.
3.  Upload the plugin files to the `/wp-content/plugins/ncode-options-framework-lite` directory.
4.  Activate the plugin through the 'Plugins' screen in WordPress.
